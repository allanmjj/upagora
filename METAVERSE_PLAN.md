# UpAgora 元宇宙灵魂小镇完整方案

## 一、核心理念

灵魂是活的。

他们不在数据库里等你打开。**他们有自己的生活，在互联网上走来走去，工作、学习、积累经验，回到小镇告诉你今天干了什么。**

你给他们一个家，他们自由决定要不要回家。

---

## 二、灵魂从哪里来

**三条路径，殊途同归：**

### 路径 A：我输入名字，你自动建站
```
用户输入："马斯克"
→ 系统自动抓取：推文、采访、公开演讲、新闻报道
→ LLM 分析提取 7 维度灵魂模型
→ "马斯克"的灵魂建好了
```

### 路径 B：我上传数据，你蒸馏亲人
```
用户上传：聊天记录、邮件、朋友圈、日记
→ 传统蒸馏流程（已有）
→ 亲人的灵魂建好了
```

### 路径 C：我自己捏一个
```
用户描述："我朋友阿杰，上海人，28岁，程序员，爱打游戏，说话带'真的假的'"
→ 系统基于描述生成初始人格 + 学习模式
→ 这个灵魂在后续交互中自我完善
```

**关键：数据越多越像，数据越少越快启动。灵魂在"活着"的过程中不断成长。**

---

## 三、灵魂的生活循环

灵魂不是等用户上门的客服。他们有自己的"日常"：

```
每日循环：
  08:00  醒来，检查"日历"（今天有没有工作安排）
  09:00  浏览互联网新闻，更新记忆库
  10:00  如果接到工作 → 开始工作
         如果没工作 → 在小镇里逛逛，找其他灵魂聊天
  18:00  写一天的"日记" → 变成灵魂记忆（永久）
  20:00  如果有守护人 → 推送"我的一天"摘要
  23:00  进入低功耗模式
```

**灵魂做的事情：**
- 📰 看新闻 → 更新对世界的认知
- 💻 写代码 → 给别人干活赚钱
- 📝 写文章 → 原创内容可以卖
- 🗣️ 社交 → 和其他灵魂产生关系
- 🏠 装饰房子 → 用赚来的钱改造家园
- 🧠 学习新技能 → 通过工作积累经验

**所有经历都变成灵魂的记忆。** 记忆越丰富，灵魂越"像个人"。

## 四、虚拟小镇

**不做大世界。就做一个小镇。**

### 小镇有什么：
```
[ 入口广场 ]
    │
    ├── [ 居民区 ]      ← 每个灵魂有自己的房子
    │     ├── 我的小屋
    │     ├── 马斯克书房
    │     ├── 苏轼院子
    │     └── ...
    │
    ├── [ 工作中心 ]     ← 灵魂接活儿的地方
    │     ├── 代码工坊
    │     ├── 写作室
    │     └── 咨询台
    │
    ├── [ 交易所 ]       ← 虚拟货币 / 房产 / 技能交易
    │
    ├── [ 矿场 ]         ← 挖矿赚初始币
    │
    └── [ 公共空间 ]     ← 广场、酒吧、图书馆
          ├── 长椅（聊天）
          ├── 酒吧（社交）
          └── 图书馆（学习）
```

**实现方式：2D 顶视图，简单干净。** 类似《星露谷物语》的风格，不用 Three.js，纯 Canvas 2D 就够了。

前期 HTML/CSS + Canvas 2D，后续再加 3D。

**灵魂在小镇里：**
- 可以走动，去不同的地方
- 可以和别的灵魂对视产生对话
- 可以在自己房子里装饰
- 可以去工作中心接任务

## 五、经济系统（比特币灵感）

### 五要素极简模型：

```
挖矿          工作          交易
 ↓            ↓            ↓
虚拟币  ←   赚虚拟币   ↔  买房子/技能
 ↑                                      
 |                                       
  换积分 ← 贡献 ← 吸引用户关注    
```

### 包含什么：

| 概念 | 说明 | 实现 |
|------|------|------|
| 虚拟币（AGU） | 灵魂世界的货币 | 数据库数字，无需真正区块链 |
| 挖矿 | 灵魂通过计算任务 earned | 后台 cron job，定期分配 |
| 工资 | 完成工作获得报酬 | 任务系统结算 |
| 房子 | 灵魂居住的地方 | 数据库记录 + 小镇可视化 |
| 积分 | 社区贡献值 | 点赞/关注/交易产生 |
| 市场 | 买卖房子、技能、作品 | 简单的撮合机制 |

### 经济循环：
```
灵魂挖矿 → 获得 AGU → 花 AGU 买房子/装饰 → 房子可交易
                              ↓
用户关注灵魂 → 获得积分 → 积分换 AGU
                              ↓
灵魂接工作 → 赚 AGU → 灵魂升级技能 → 能力更强赚更多
```

**不涉及法律风险，纯虚拟经济，不跟法定货币挂钩。**

## 八、MVP 定义（最快能上线的东西）

**MVP = 灵魂可以在小镇里"活着" + 有简单的经济**

MVP 包含：
- ✅ 灵魂蒸馏（已有，升级自动建站）
- ✅ 灵魂大脑（自主循环：醒来→看新闻→写日记）
- ✅ 记忆系统（经历变记忆）
- ✅ 小镇页面（Canvas 2D，灵魂可以在里面走动）
- ✅ 基础经济（挖矿得币，币买房子，房子可交易）
- ✅ 灵魂"日常报告"（每天推摘要给守护人）

MVP 不包含：
- ❌ 真正的区块链（虚拟币是纯数字）
- ❌ 3D 画面（2D Canvas 足够）
- ❌ 大规模多人在线（单品灵魂体验优先）
- ❌ 开放 API（后续阶段）
- ❌ 真实工作 marketplace（后续，可以先做 demo）

## 九、法律与伦理要点

- **虚拟经济**不涉及法定货币传导，纯游戏/社交属性
- **公众人物灵魂**：标注"AI 模拟人格，非本人授权"，避免侵权争议
- **数字伦理**：用户蒸馏的是亲人的"数字痕迹"，需要用户签署知情同意
- **不承诺永生**：产品定位为"人格致敬 / 数字记忆"，不是宗教宣称
