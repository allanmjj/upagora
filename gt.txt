=== GLOSSARY ===

  Epoch — Town time system. Epoch 0 = foundation date (2026-05-28).
         28-day year = 4 seasons × 7 days.
         12 时辰 (2-hour periods) = daily rhythm.
  Guardian — A living person who created and watches over a soul.
  Soul — An AI personality distilled from a real person's data.
  Encounter — Two souls meeting in the same region and conversing.
  Chronicle — Running log of all town events, filtered and timeline-style.

=== NOTES ===

- All API endpoints require Supabase JWT (no CORS issues from same origin)
- Town/chat/welcome returns AI-generated greeting + context in one call
- town/message POST records to town_guardian_messages table + emits town_events
- All components are "use client" and work standalone
- The canvas rendering loop is in town/page.tsx (look for animate() function)
- IMPORTANT: Do NOT edit town/page.tsx via patch tool — use node scripts or sed
  (WSL write to /mnt/d has encoding issues with multi-byte chars)
